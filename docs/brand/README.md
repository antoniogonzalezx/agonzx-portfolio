# Brand system — two tracks, one DNA

Antonio operates two distinct brands. They share the `{x}` monogram, the
typography stack and the easing curve. Everything else — palette, voice,
audience, language, channels — is intentionally different.

| | **agonzx** | **axlab** |
|---|---|---|
| Type | Personal brand | Studio brand |
| Identity | Senior iOS engineer | Software a medida studio |
| Audience | Tech recruiters, product company hiring managers, indie users | Spanish SME owners, gerentes de pyme |
| Voice / language | English, professional but human | Spanish "tú", direct, anti-jerga |
| Mode | Dark | Light editorial |
| Domain | `agonzx.dev` | `axlab.es` |
| Email | `antoniogonzalezvaldepenas.jobs@gmail.com` | `contacto@axlab.es` |
| Wordmark | `agonz{x}` | `a{x}lab` |
| Channels | LinkedIn, GitHub, App Store, Twitter (when active) | WhatsApp Business, dedicated phone, local outreach |

The single visual thread is the `{x}` brace-x monogram. It appears in
both brands. Everything else lives in its own column above. Never mix
contact info, voice or palette between tracks.
